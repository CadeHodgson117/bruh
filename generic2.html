Unity Project: Sidescrolling shooter

This project initially began as a framework that I worked on where I had the option of making 1 of 3 types of top down shooter. Of the 3 options available to me, I ended up going with the vertical shooter framework which I then modified and personalised using various C# scripts and other assets within Unity to make it more unique and in-depth.

Another Unity project I worked on was a Unity recreation of the classic game "Marble Madness" although very stripped down and bare. Given a few weeks I was able to develop this framework into something more interesting and fun to play and was a great experience for all the things I learned about Unity and its range of tools.
