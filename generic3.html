Game Maker 2D Project: Metroidvania prototype

This project was one I started back in my games design college course and ended up being a major starting point for myself as a programmer. While Game Maker 2D sees very fringe use throughout the industry, it was an immense help in learning how the general logic of games programming works, with things such as if statements, conditionals, x and y positions etc. While this project is not that in depth, it remains my most personal project as it was entirely built from the ground up by myself and was a major turning point in my games production journey.
